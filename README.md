# FakeBerries

FakeBerries — учебный мини-маркетплейс на Flask. Приложение позволяет администратору управлять карточками товаров, переносить их в корзину и оформлять «покупку» для тренировки пользовательских сценариев.

## Страницы

- **Главная (`/`)** — карточки товаров сеткой 3×3 (максимум 9 элементов на страницу). Для каждого товара доступны действия «В корзину» и «Удалить».
- **Корзина (`/basket`)** — список всех товаров со страницы 2. Кнопка «Удалить из корзины» возвращает позицию на главную, а «Оформить покупку» очищает корзину и удаляет товары.
- **Добавление (`/product/add`)** — форма загрузки изображения, ввода названия и цены. Доступна только из каталога.
- **Вход (`/login`)** — единственная учётная запись admin/admin123.

## Технологии

- Flask + Flask-Login
- SQLAlchemy + SQLite
- Bleach для санитизации полей
- HTML5 + собственные стили/скрипты без сторонних UI-библиотек

## Быстрый старт

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Приложение будет доступно на `http://localhost:5000`.

## Поведение корзины

1. На главной хранится до 9 товаров. Остальные попадают на вторую страницу каталога (`?page=2`).
2. Кнопка «В корзину» изменяет статус товара без дублирования записей.
3. В корзине можно удалить товар (возвращается на главную) или оформить покупку (товары и изображения удаляются).

## Авторские права

Проект создан в образовательных целях. Все изображения добавляются пользователем и хранятся локально.
